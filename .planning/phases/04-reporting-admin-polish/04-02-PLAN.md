package com.pcori.platform.domain.user;

**UserService.java**:
- `@Service @Transactional @RequiredArgsConstructor @Slf4j`
- Inject: `UserRepository userRepository`, `RoleRepository roleRepository`, `PasswordEncoder passwordEncoder`
- `createUser(CreateUserRequest req)`: validate username/email uniqueness (throw `DomainExceptions.ConflictException` for duplicate); hash password with `passwordEncoder.encode()`; resolve each role name via `roleRepository.findByName()` (throw `ResourceNotFoundException` if role not found); build and save User; return `UserResponse`
- `updateUser(UUID id, UpdateUserRequest req)`: find user or throw `ResourceNotFoundException`; patch only non-null fields; if `req.roles()` is non-null and non-empty, resolve and replace roles; if non-null but empty, throw `InvalidRequestException("At least one role is required")`; save and return `UserResponse`
- `deactivateUser(UUID targetId, UUID currentUserId)`: **DEACTIVATION GUARD** — if `targetId.equals(currentUserId)` throw `InvalidRequestException("You cannot deactivate your own account")`; find user; set `user.setActive(false)`; save; return `UserResponse`
- `reactivateUser(UUID targetId)`: find user; set `user.setActive(true)`; save; return `UserResponse`
- `deleteUser(UUID id)`: soft-delete by setting `user.setDeletedAt(Instant.now())`; save (the `@SQLRestriction("deleted_at IS NULL")` on User entity will hide it from future queries)
- `findById(UUID id)`: return `UserResponse` or throw `ResourceNotFoundException`
- `listAll(Pageable pageable)`: `userRepository.findAll(pageable)` mapped to `PagedResponse<UserResponse>`
- `listActive()`: `userRepository.findAll(where(statusSpec(true)))` as `List<UserResponse>`
- `searchUsers(String q, String role, Boolean active, Pageable pageable)`: build `Specification` using `UserSpecification`; call `userRepository.findAll(spec, pageable)`; return `PagedResponse<UserResponse>`
- Private `toResponse(User u)`: maps User to UserResponse including `u.getRoleNames()`

Add any missing exception types to `DomainExceptions.java` if `ConflictException` or `InvalidRequestException` don't exist yet (check the file first and add only what's missing).

Create `UserController.java` exposing the following endpoints from TechArch API spec. Use `@PreAuthorize("hasRole('ADMIN')")` on all write endpoints; GET endpoints may allow broader access.

**Endpoints to implement (exact paths from TechArch):**

```
GET    /api/users                                    → listAll(Pageable) → 200 PagedResponse<UserResponse>
GET    /api/users/{id}                               → findById(UUID) → 200 UserResponse
POST   /api/users                                    → createUser(@Valid CreateUserRequest) → 201 UserResponse
PUT    /api/users/{id}                               → updateUser(UUID, @Valid UpdateUserRequest) → 200 UserResponse
DELETE /api/users/{id}                               → deleteUser(UUID) → 204 No Content
PATCH  /api/users/{id}/status                        → toggleStatus (body: {"active": boolean}) → 200 UserResponse
GET    /api/users/search?q=&role=&status=&page=&size= → searchUsers → 200 PagedResponse<UserResponse>
GET    /api/users/active                             → listActive() → 200 List<UserResponse>
```

**PATCH /api/users/{id}/status** body: accept `record StatusRequest(boolean active)`. If `active=false`, delegate to `userService.deactivateUser(id, currentUserId)` (pass `@AuthenticationPrincipal User principal` to get currentUserId). If `active=true`, delegate to `userService.reactivateUser(id)`.

**Controller structure:**
```java
@RestController
@RequestMapping("/api/users")
@RequiredArgsConstructor
@Slf4j
public class UserController {
    private final UserService userService;

    @GetMapping
    @PreAuthorize("hasRole('ADMIN')")
    public ResponseEntity<PagedResponse<UserResponse>> listAll(Pageable pageable) { ... }

    @GetMapping("/active")
    // No role restriction — used by other domains to resolve user IDs
    public ResponseEntity<List<UserResponse>> listActive() { ... }

    @GetMapping("/search")
    @PreAuthorize("hasRole('ADMIN')")
    public ResponseEntity<PagedResponse<UserResponse>> search(
        @RequestParam(required=false) String q,
        @RequestParam(required=false) String role,
        @RequestParam(required=false) Boolean active,
        Pageable pageable) { ... }

    @GetMapping("/{id}")
    @PreAuthorize("hasRole('ADMIN')")
    public ResponseEntity<UserResponse> getById(@PathVariable UUID id) { ... }

    @PostMapping
    @PreAuthorize("hasRole('ADMIN')")
    public ResponseEntity<UserResponse> create(@Valid @RequestBody CreateUserRequest req) { ... }

    @PutMapping("/{id}")
    @PreAuthorize("hasRole('ADMIN')")
    public ResponseEntity<UserResponse> update(@PathVariable UUID id, @Valid @RequestBody UpdateUserRequest req) { ... }

    @PatchMapping("/{id}/status")
    @PreAuthorize("hasRole('ADMIN')")
    public ResponseEntity<UserResponse> toggleStatus(
        @PathVariable UUID id,
        @RequestBody StatusRequest req,
        @AuthenticationPrincipal User principal) { ... }

    @DeleteMapping("/{id}")
    @PreAuthorize("hasRole('ADMIN')")
    public ResponseEntity<Void> delete(@PathVariable UUID id) { ... }
}
```

The `User principal` cast works because `User extends UserDetails` (same pattern as `ClassificationController` which uses `User instanceof` cast — see existing code).

Use `Pageable` with `@PageableDefault(size=20)` on paginated endpoints.
