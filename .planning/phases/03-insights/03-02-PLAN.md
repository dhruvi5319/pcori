Append the following CSS to the END of frontend/src/app/globals.css. Do NOT remove or modify existing CSS (dash-flow keyframe, existing tokens, etc.). Only append:

```css
/* ================================================================
   Phase 3 — Skeleton shimmer loading system
   Replaces animate-pulse for all Phase 3 skeleton components.
   UI-SPEC §Loading Skeleton System (user decision — locked)
   ================================================================ */
@keyframes shimmer-slide {
  0%   { background-position: -200% 0; }
  100% { background-position:  200% 0; }
}

.skeleton-shimmer {
  background: linear-gradient(
    90deg,
    var(--skeleton-base) 25%,
    var(--skeleton-highlight) 50%,
    var(--skeleton-base) 75%
  );
  background-size: 200% 100%;
  animation: shimmer-slide 1.5s ease-in-out infinite;
}

/* Light mode skeleton tokens */
:root {
  --skeleton-base:      #F1F3F6;
  --skeleton-highlight: #E4E7ED;
}

/* Dark mode skeleton tokens */
.dark {
  --skeleton-base:      #1F1F1F;
  --skeleton-highlight: #2A2A2A;
}

/* ================================================================
   Phase 3 — Chart tooltip card
   Applied to custom Recharts tooltip wrapper div.
   UI-SPEC §Color > Recharts Custom Tooltip Card
   ================================================================ */
.chart-tooltip-card {
  background: #FFFFFF;
  border-radius: 8px;
  box-shadow: 0 4px 16px rgba(0,0,0,0.12), 0 2px 4px rgba(0,0,0,0.08);
  border: 1px solid #E5E7EB;
  padding: 8px 16px;
  font-family: 'Geist Sans', sans-serif;
  font-size: 14px;
}

.dark .chart-tooltip-card {
  background: #1F1F1F;
  border-color: #2A2A2A;
}

/* ================================================================
   Phase 3 — Dashboard KPI drag-to-reorder (UI-SPEC §Interaction)
   ================================================================ */
.kpi-drag-ghost {
  opacity: 0.6;
  transform: scale(1.02);
  border: 2px dashed #1D4ED8;
  box-shadow: 0 8px 32px rgba(0,0,0,0.15);
  border-radius: 12px;
  cursor: grabbing;
}

.kpi-drop-target-placeholder {
  border: 2px dashed rgba(29, 78, 216, 0.3);
  border-radius: 12px;
  background: rgba(29, 78, 216, 0.05);
  height: 120px;
}

/* ================================================================
   Phase 3 — Notification drawer slide animation (UI-SPEC §Animation)
   ================================================================ */
.notification-drawer {
  transform: translateX(100%);
  transition: transform 0.25s ease-out;
}

.notification-drawer[data-state="open"] {
  transform: translateX(0);
}

.notification-drawer[data-state="closed"] {
  transform: translateX(100%);
}

/* ================================================================
   Phase 3 — Analytics chart loading pulse (UI-SPEC §Animation)
   ================================================================ */
.analytics-chart-section {
  transition: opacity 0.3s ease;
}

.analytics-chart-section[data-loading="true"] {
  opacity: 0.5;
  pointer-events: none;
}
```
