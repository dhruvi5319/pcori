**frontend/src/hooks/useDashboard.ts:**
- `useDashboardMetrics(startDate?, endDate?)` — queries GET /api/dashboard/metrics or /api/dashboard/metrics/range; staleTime: 30000; refetchInterval: 30000 (always polls for live updates)
- `useDashboardConfiguration()` — queries GET /api/dashboard/configuration; staleTime: Infinity
- `useSaveDashboardConfiguration()` — mutation for PUT /api/dashboard/configuration; invalidates configuration cache on success + shows "Layout saved" green sonner toast
- `useDeleteDashboardConfiguration()` — mutation for DELETE; invalidates + shows "Layout reset to default" muted toast
- `useRecentClassifications()` — queries GET /api/classifications?size=10&sort=uploadedAt,desc; staleTime: 30000 (reuse existing useClassifications hook or make separate hook)
